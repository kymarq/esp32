Documentación del distribuidor (Sunfounder) para los componentes del kit de iniciación

Suficientemente detallado pero sencillo a la vez.
Permite entender rapidamente el funcionamiento y partes de cada elemento, conexión, posibles aplicaciones,etc.

Sunfounder componentes.

Guía rápida de Micropython para 32 proyectos de iniciación

Guía para cada uno de los 32 proyectos propuestos por el kit de iniciacion para aprender. De cada proyecto se incluye el codigo de Micropython, el esquema eléctrico y diagramas para visualizar el cableado de los componentes, pines, breadboard,etc.

Sunfounder Micropython.

Thonny. Editor para desarrollo en Python

Thonny es el software recomendado por la propia Sunfounder para los proyectos de ESP32 en Micropython. Interfaz sencilla y limpia. En general resulta bastante mas agradable la configuración y el manejo en comparacion con IDLE o upycraft.

Coolors.co: Generador de paletas de color

Paleta utilizada para el fondo y los enlaces circulares.


Stack Overflow: Javascript

Hilo donde el usurario bdesham comparte una forma sencilla para generar colores aleatorios a partir de una lista en Javascript.
Adaptado con mi paleta seleccionada para cambiar el color de fondo de la página index cada vez que se recarga la página.


Generador de coordenadas para image map

Generador de coordenadas para seleccionar el area de una imagen de manera rapida y obtener las coordenadas del image map.
Usado en el index para crear el enlace con image map de la zona central de la imagen del ESP32.


Affinity Photo y Affinity Designer

AFFINITY Software de Pago. La mejor alternativa sin suscripcion a los programas Adobe Photoshop/Adobe Illustator.
Utilizados para crear la imagen de fondo del index y el icono de ESP32 en formato svg para la pestaña del navegador y el boton para volver al inicio desde otra sección.


WOKWI

WOKWI Para simular el código antes de probarlo en la placa.


Micro REPL para Android

Google Play Store IDE para Micropython en Android. Instalando esta App en el móvil, se puede conectar directamente la placa ESP32 al móvil por medio del cable usb y subir scripts sin necesidad de PC.


Deepseek AI

Deepseek Utilizado sobre todo para solucionar errores y ayudarme a maquetar/alinear elementos en el CSS de forma ordenada sin volverme loco.
